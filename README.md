# CONTROL ESCOLAR

## Descripción
Sistema de control escolar para gestionar alumnos, profesores, 
administrativos, directores de carrera y secretarios académicos.

## Objetivo
Desarrollar una aplicación web que permita llevar el control 
escolar de una institución educativa.

## Pila Tecnológica
- Laravel (PHP)
- MySQL
- HTML / CSS / JavaScript
- Git / GitHub
